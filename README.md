# Born2beRoot

#### 概要
  * VirtualBoxVMを使用し、Debian(Linux)をセットアップする課題<
  
#### 使用環境
  * VirtualBoxVM
  * Debian(Linux)
  
#### 設定内容
  * userのパスワードポリシー設定
    * common-passwordファイル
    * login.defsファイル

  * sudo権限の設定
    * visudoファイル
  
  * ufwの設定
    * ポート4242のみ設定

  * sshの設定
    * ポート4242で接続できるよう設定

  * cronの設定
    * 10分間隔で、monitoring.shが動作するよう設定

  * シェルファイルの作成
    * monitoring.shファイル

  * ユーザー作成
  
  * グループ作成、追加

  * hostnameの変更
